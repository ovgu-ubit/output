Informationen zur Demo-Version

Allgemeine Informationen

Diese Demo-Version dient der unverbindlichen, öffentlichen Erprobung aller Funktionen der Publikationskostenmonitoring-Lösung Output. Diese wird auf GitHub entwickelt.

Alle enthaltenen Informationen, Daten und Inhalte sind öffentlich verfügbar oder fiktiv und dienen ausschließlich Demonstrationszwecken. Es handelt sich nicht um reale Vertrags- und Finanzdaten.

Die Zugangsdaten für die Demo-Instanz können per E-Mail an sbosse@ovgu.de angefordert werden. Bitte geben Sie keine (echten) personenbezogenen, vertraulichen, finanziellen oder produktiven Daten ein.

Die Anwendung und die Daten werden in regelmäßigen Abständen zurückgesetzt, um die Demo-Instanz für alle Nutzer*innen zugänglich und funktionsfähig zu halten.
